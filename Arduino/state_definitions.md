# State definitions Quantum Dice

The Quantum Dice is controlled via 2 state type: 
1. states defined as a Finite State Machine
2. the dice States

## Finite State Machine (FSM) states)

The following states are defined:
0.	IDLE
1.	INITSINGLE
2.	INITENTANGLED_AB1
3.	INITENTANGLED_AB2
4.	INITSINGLE_AFTER_ENT
5.	WAITFORTHROW
6.	THROWING
7.	INITMEASURED
8.	LOWBATTERY
9.	CLASSIC_STATE

Changes of the states is initiated by conditions. The flow of states is defined in a state diagram.
